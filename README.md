# 📊 Social Media Analytics Dashboard (Microsoft Excel)

## Overview

This project is an interactive **Executive Social Media Analytics Dashboard** built entirely in **Microsoft Excel** using **Pivot Tables, Pivot Charts, Slicers, Timelines, Conditional Formatting, and Dynamic KPI Cards**.

The dashboard analyzes **12,000 social media posts** published across multiple platforms over a one-year period to help marketing teams monitor campaign performance, audience engagement, content effectiveness, geographic reach, and sentiment trends.

The goal of this project is to demonstrate practical **Data Analytics**, **Business Intelligence**, and **Dashboard Design** skills using Excel.

---

# Project Objective

Modern marketing teams generate massive amounts of social media data.

This dashboard transforms raw marketing data into meaningful business insights that help answer questions such as:

* Which platform generates the highest engagement?
* Which campaigns perform best?
* Which brands receive the highest audience interaction?
* Which countries contribute the most engagement?
* Which content types perform better?
* How does engagement change over time?
* Which emotions dominate customer responses?
* Which campaign phases perform best?
* Which topics attract the highest engagement?

---

# Dataset

The dataset contains **12,000 social media posts** collected over a 12-month period.

Each record includes:

* Platform
* Country
* Location
* Language
* Brand
* Product
* Campaign
* Campaign Phase
* Content Type
* Topic Category
* Likes
* Shares
* Comments
* Impressions
* Engagement Rate
* Sentiment Score
* Sentiment Label
* Emotion Type
* Posting Time

---

# Data Cleaning

The raw dataset was cleaned using Microsoft Excel.

Cleaning steps included:

* Removed duplicate records
* Corrected inconsistent text values
* Standardized country and platform names
* Fixed formatting issues
* Validated numerical columns
* Verified engagement metrics
* Added derived columns including:

  * Posting Hour
  * Country
  * Content Type
* Organized the cleaned dataset for Pivot Table analysis
